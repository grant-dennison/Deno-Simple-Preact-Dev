# Deno Simple Preact Dev

This project is intended to demonstrate a super simple Preact web application
with minimal dev server overhead using Deno.

## Prerequisites

[Deno](https://deno.com) must be [installed](https://docs.deno.com/runtime/getting_started/installation/).

```sh
# Linux
curl -fsSL https://deno.land/install.sh | sh
# Windows
# irm https://deno.land/install.ps1 | iex
```

## Quick Start

```sh
deno task dev
```

Then visit your web application at <http://localhost:8000>!
